Uso de la función wsa_spectrum

Esta función estima el espectro de energía de una secuencia de datos. El espectro calculado corresponde a la densidad espectral de potencia unilateral (frecuencias positivas). El espectro se calcula mediante el método de periodograma medio, siguiendo la metodología de Welch-Barlett.
La función permite ser utilizada de forma general para cualquier señal, cuyo espectro resultante tendrá unidades de [unidad de X]^2 / Hz. Además, permite el cálculo del espectro a partir de una señal de presión que deba ser corregida por Kp, especificando 'InputType' 'pressure'.

Inicialización

clc; clear; close all;
 
addpath(genpath('..\..\'));

Datos de entrada

Para mostrar el uso de la función, en la carpeta \toolbox\example_data se colocó un struct con datos reales de una campaña de medición de AWAC.
data = load('burst_data.mat');
Se extrae la información necesaria para estimar espectros basados en señales de presión y AST.
  • AST: señal de superficie libre (preprocesada mediante despike y filtrado) [m].
  • Pressure: señal de presión (preprocesada mediante filtrado) [dBar].
  • fs: frecuencia de muestreo [Hz].
  • ast_mean: distancia media a la superficie desde el sensor AWAC, se utiliza para estimar la profundidad [m].
  • mounting_height: altura de montaje del sensor AWAC [m].
  • h: profundidad del lecho marino [m].
  • z_p: distancia de la superficie libre al sensor (negativo hacia abajo) [m].
  • Kp_min: umbral de Kp mínimo empleado.
AST = data.burst_data.processed.ast(:, 1);
Pressure = data.burst_data.processed.pressure;
 
fs = data.burst_data.general.fs;
ast_mean = data.burst_data.general.ast_mean;
mounting_height = data.burst_data.general.mounting_height;
 
h = ast_mean + mounting_height;
z_p = -ast_mean;

Espectro basado en AST

Parámetros por defecto

Por defecto el espectro se calcula con 16 grados de libertad y ventana de Hanning
[out_Spec, info_Spec] = wsa_spectrum(AST, fs);
f = out_Spec.f;
S = out_Spec.S;
 
plot_spec(f, S, info_Spec)

Indicando los grados de libertad

Al aumentar los grados de libertad, disminuye la varianza del espectro.
 
[out_Spec, info_Spec] = wsa_spectrum(AST, fs, ...
'DoF', 64);
f = out_Spec.f;
S = out_Spec.S;
 
plot_spec(f, S, info_Spec)
 

Espectro basado en presión

Parámetros por defecto

El espectro de presión requiere que se indiquen los parámetros un, z_p y h; caso contrario se presenta un error. Por defecto el valor de Kp mínimo corresponde a 0.2.
 
[out_Spec, info_Spec] = wsa_spectrum(Pressure, fs, ...
'InputType', 'pressure', ...
'un', 'dBa', ...
'z_p', z_p, ...
'h', h);
f = out_Spec.f;
S = out_Spec.S;
 
plot_spec(f, S, info_Spec)
 

Indicando los grados de libertad

[out_Spec, info_Spec] = wsa_spectrum(Pressure, fs, ...
'InputType', 'pressure', ...
'un', 'dBa', ...
'z_p', z_p, ...
'h', h, ...

Indicando el umbral de Kp

 
[out_Spec, info_Spec] = wsa_spectrum(Pressure, fs, ...
'InputType', 'pressure', ...
'DoF', 64, ...
'un', 'dBa', ...
'z_p', z_p, ...
'h', h, ...

f = out_Spec.f;
S = out_Spec.S;
 
plot_spec(f, S, info_Spec)
